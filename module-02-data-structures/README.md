# Module 02: Python Data Structures

## Objective
Understand how to use lists, sets, tuples, and functions in Python.

## Topics Covered
- List operations and duplicate removal  
- Set operations (union, intersection, difference)  
- Tuple unpacking and comparison  
- Functions with parameters and return values  

## Key Learnings
- Writing reusable functions  
- Handling collections of data efficiently  
- Understanding mutability vs immutability

# Module 03: Object-Oriented Programming

## Objective
Understand how to structure programs using classes and object-oriented principles.

## Topics Covered
- Class creation and constructors  
- Inheritance (single, multilevel, hierarchical)  
- Encapsulation (private attributes, getters/setters)  
- Method overriding and overloading  

## Key Learnings
- Writing modular and reusable code  
- Modeling real-world problems using classes  
- Managing data access and behavior effectively
